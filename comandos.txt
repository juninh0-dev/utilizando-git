-> Pra entrar no seu usuario git

git config --global user.name "Altair Preto de Godoi Junior"
git config --global user.email altair.godoi@fatec.sp.gov.br

-> inicializa um repositório git (utilizando apenas 1 vez)
git init

-> verificar o status de repositório
git status

-> Adicionar os novos arquivos no git para versionar
git add .

-> Salva uma versão dos arquivos
git commit -m "Por que estou salvando???"

-> Envia os commits para o github
git push

-> Baixa a atualização dos arquivos do github para uma pasta já existente
git pull
